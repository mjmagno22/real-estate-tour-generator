# Real Estate Tour Generator

Create immersive 3D virtual real estate tours in minutes. Upload floor plans, add furniture, get shareable links.

## Features

- 📄 Upload floor plan images
- 🎨 Customize 3D environments
- 🏠 Generate virtual tours
- 💰 $19 per tour pricing model

## Tech Stack

- React + Vite
- Three.js for 3D rendering
- Vercel for deployment
- Stripe for payments (coming soon)

## Quick Start

```bash
npm install
npm run dev
```

## Monetization

- **Agent Plan**: $19 per tour
- **Broker Plan**: $99/month (unlimited tours)
- **Enterprise Plan**: Custom pricing

Built for real estate agents to create virtual tours quickly.
